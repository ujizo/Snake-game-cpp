# 🐍 Snake Game

Простая реализация классической игры "Змейка" на C++ для консоли Windows.

## 🎮 Особенности

- Цветной консольный интерфейс
- Три уровня сложности
- Система счета и длины змейки
- Управление WASD
- Меню выбора сложности

## 🎯 Как играть

### Управление
- **W** / **A** / **S** / **D** - движение вверх/влево/вниз/вправо
- **X** - выход из игры
- **Стрелки** - навигация в меню
- **Enter** - выбор в меню

### Уровни сложности
- 🌱 **Easy** - медленная скорость (идеально для новичков)
- ⚡ **Medium** - средняя скорость
- 🔥 **Hard** - высокая скорость (для профи)

### Правила
- 🍎 Съешьте фрукт, чтобы вырасти и получить +10 очков
- 💀 Столкновение со стеной или собственным хвостом = Game Over
- 📏 Длина змейки = начальная длина (1) + съеденные фрукты

## 🛠️ Сборка проекта

### Visual Studio
1. Создайте пустой проект C++
2. Добавьте все `.cpp` и `.h` файлы
3. Настройте кодировку проекта на "Multibyte Character Set"
4. Скомпилируйте (Ctrl+F5)

### Требования
- Windows OS
- Visual Studio (или любой компилятор с поддержкой WinAPI)
- Консоль с поддержкой цветов

## 📁 Структура проекта

```
SnakeGame/
├── Main.cpp           # Точка входа
├── Game.cpp           # Логика игры
├── Game.h             # Заголовок игры
├── ConsoleUtils.cpp   # Утилиты консоли
├── ConsoleUtils.h     # Заголовок утилит
```

## 🎨 Особенности реализации

- Плавное управление без задержек
- Защита от разворота на 180 градусов
- Умная генерация фруктов (не появляются на змейке)
- Красивый цветной интерфейс
- Отображение текущего счета и длины

## 📝 Примечания

- Игра использует Windows API для управления консолью
- Все настройки (размер поля, скорость) можно изменить в `Game.h`
- Приятного времяпрепровождения! 🎉
